# Security Policy

## Supported versions

Only the latest release receives security updates.

| Version | Supported |
|---|---|
| 1.2.x | ✅ |
| 1.1.1 | ❌ |
| 1.0.0 | ❌ |

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

To report a vulnerability, use one of the following:

- **GitHub private disclosure** — use the [Report a vulnerability](https://github.com/scr0ols/SoundTweaks/security/advisories/new) button on the Security tab
- **Email** — contact joaosilva98.work@gmail.com

Please include as much detail as possible: steps to reproduce, potential impact, and your environment (mod version, Minecraft version, Fabric Loader version).

You can expect an initial response within **72 hours**. If the vulnerability is confirmed, a fix will be prioritised for the next release.
